# Bimto3dPrint

## RU

Bimto3dPrint автоматизирует экспорт архитектурной оболочки из Revit для 3D печати.

### Быстрый старт
1. Выберите конфигурационный пресет в `Config/Presets/`.
2. Экспортируйте IFC из Revit (плагин в `RevitPlugin/`).
3. Запустите Python пайплайн для обработки IFC.

## EN

Bimto3dPrint automates exporting the architectural shell from Revit for 3D printing.

### Quick start
1. Pick a preset in `Config/Presets/`.
2. Export IFC from Revit (plugin in `RevitPlugin/`).
3. Run the Python pipeline to process the IFC.
