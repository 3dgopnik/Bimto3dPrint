# Политика агента

## Общие правила
- Общение в чате — на русском.
- Код/диффы/команды публикуются только в PR, не в чате.
- Изменения должны быть небольшими и фокусными.
- CI обязателен (линтер → форматирование → сборка → тесты).
- Идеи фиксируются в `docs/IDEAS.md` (1–2 идеи на каждое изменение).

## Документация
- Документация двуязычная: `docs/ru/` (приоритет) и `docs/en/`.
- При изменениях обновлять обе версии.

## Безопасность
- Не добавлять секреты в репозиторий.
- Не коммитить бинарные файлы.

## Процесс PR
- Использовать шаблон PR.
- Заполнять все разделы, включая Risks, Rollback и Instruction.
