# Руководство пользователя

## Установка плагина Revit
1. Соберите проект `RevitPlugin` в Release.
2. В `bin/Release` будет сгенерирован `Bimto3dPrint.addin` с абсолютным путём к DLL.
3. Скопируйте в папку Addins вашей версии Revit:
   - `Bimto3dPrint.dll`
   - `Bimto3dPrint.addin`
   - папку `Config/Presets/Revit`
4. Создайте `bimto3dprint.settings.json` рядом с `.addin` (шаблон: `RevitPlugin/bimto3dprint.settings.json.template`).

## Использование
1. Откройте модель Revit.
2. Запустите команду **Bimto3dPrint Export**.
3. Выберите пресет, IFC версию, папку вывода и формат результата.
4. При необходимости включите **Run Python pipeline after export**.
5. Запустите экспорт IFC и последующую обработку.

> Здесь будут добавлены скриншоты диалога (плейсхолдеры без бинарных файлов).

## Python-процессор
В составе пайплайна доступны вспомогательные классы:
- `IFCLoader` — загрузка IFC и отбор элементов по категориям.
- `MeshOptimizer` — подготовка сетки к 3D-печати (оболочка, утолщение, сглаживание, валидация).

## Настройки
- Категории: включаемые/исключаемые категории оболочки.
- Параметры упрощения: целевое число треугольников, допуск.
- Печать: масштаб, толщина стенок, минимальный размер элементов.

## Troubleshooting
- Экспорт пустой: проверьте активный 3D вид и список категорий.
- Большой размер файла: увеличьте упрощение или исключите мелкие детали.

## FAQ
**Q:** Как исключить мебель?
**A:** Добавьте `BuiltInCategory.OST_Furniture` в `exclude`.
