# Быстрый старт

> Этот раздел будет обновлён после реализации базового пайплайна.

Планируемый сценарий:
1. Запустить Revit-плагин и выбрать пресет фильтрации оболочки.
2. Экспортировать IFC из подготовленного 3D вида.
3. Запустить Python-обработчик для очистки и подготовки к печати.
4. Открыть итоговый файл в 3ds Max.
