# IFC_BuildingEnvExtractor — краткий анализ

> Примечание: репозиторий `tudelft3d/IFC_BuildingEnvExtractor` недоступен из текущей среды (HTTP 403). Анализ основан на типовых практиках обработки IFC. После восстановления доступа требуется сверка с исходным кодом.

## Ожидаемая архитектура

1. Загрузка IFC (`ifcopenshell.open`).
2. Отбор категорий оболочки (стены, крыши, перекрытия и т.д.).
3. Тесселяция геометрии (`ifcopenshell.geom`).
4. Агрегация мешей в один.
5. Извлечение оболочки (voxel/alpha/convex hull).
6. Пост‑обработка (ремонт, упрощение, масштаб).
7. Экспорт (STL/OBJ/FBX).

## Адаптация для 3D‑печати

- Масштабирование под объём принтера.
- Упрощение с контролем формы.
- Гарантия watertight и корректных нормалей.
- Валидация и логирование перед экспортом.
